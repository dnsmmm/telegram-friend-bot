# Sardor Bot v2

Telegram bot with OpenRouter GPT-4.1 Mini, PostgreSQL memory, reminders, and proactive messages.

## Railway variables

Required:

- TELEGRAM_BOT_TOKEN
- OPENROUTER_API_KEY
- DATABASE_URL

## Commands

- /start
- /remember text
- /memory
- /reset
- /proactive_on
- /proactive_off
- /napishi 10
